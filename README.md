## Installation

```
$ git clone https://github.com/nuruddinsayeed/django_real_estate_project.git
```
```
$ cd (enter the downloaded directory)
```
```
$ docker build .
```
```
$ docker-compose build
```
```
$ docker-compose up
```
```
$docker-compose run app sh -c "python manage.py runserver"
```
